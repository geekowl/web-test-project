과제

조건: bootstrap을 이용해(선택) 헤더,네비게이션, 컨텐트, 푸터가 있는 페이지 틀을 만든다.

+네비게이션에 4개 페이지를 넣는다.

1페이지 : 별찍기

1페이지 : 별찍기 에서는 테이블을 이용해서 삼각형 모양 4개의 별을 찍는다.

(*테투리를 없애고 텍스트 상태로)

2페이지 : 인사

2페이지 : 인사 input에 이름을 입력한 뒤 '인사'버튼을 누르면 밑에 있는 칸에 '(이름)님 안녕하세요?'라는 메시지가

입력되도록 버튼을 누를 때마다 메시지가 계속 늘어나야 한다.

3페이지 : 체크

3페이지 : 체크 체크박스에 5개의 숫자를 매기고 확인버튼을 눌렀을 때 3번째 체크박스가 선택되어 있으면 잘 되어있어야 한다는 알림창, 선택되어있지 않으면 안 되어 있다는 알림창을 띄어야 한다.

4페이지 : 홀짝

4페이지 : 홀짝 5개의 이미지에 같은 클래스를 적용 시킨다.

(*pseudo selector를 이용해서 홀수번째 이미지와 짝수번째 이미지에 다른 스타일을 적용)

8/6 추가

소스트리를 다시 깔고 github repository에 지금까지 했던 작업물을 전부 push

아래 항목을 하나씩 할 때마다 commit한 후 push하기

  • 지금까지 썼던 자바스크립트 코드를 전부 jQuery로 다시 작성해본다.
  • 자바스크립트 코드를 별개의 파일로 뺀다.
  • 4페이지 스타일을 테두리 색깔로
  • 4페이지에 '추가' 버튼을 만들고 추가 버튼을 누를 경우 5번 이미지 뒤에 다시 1번부터 5번까지 순서대로 이미지가 추가된다.
  • 4페이지에 있는 이미지를 클릭하면 클릭된 이미지가 사라진다.
